# Semana 02 / Aula 02

**Qual ferramenta de versionamento vocês escolheriam para a empresa mencionada no contexto.**
R: O recomendado para esse cenário seria utilizar o Git como ferramenta de versionamento.

**Justifiquem suas escolhas com base no que foi discutido na aula.**
R: Permite que cada desenvolvedor tenha uma cópia completa do repositório localmente, o que melhora a performance e a capacidade de trabalhar offline. Além disso, seu modelo de ramificação é leve e eficiente, facilitando o desenvolvimento paralelo de funcionalidades e a integração contínua.
