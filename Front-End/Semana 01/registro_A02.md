**Editores de Texto x Editores de Código**
Quando utilizamos um editor de texto simples para criar um web, encontramos várias dificuldades, como a falta de cores no código, ausência de ajuda automática e dificuldade para encontrar erros. Já os editores de código profissionais facilitam muito esse processo, pois eles deixam de forma mais organizada o código visualmente, usam cores para diferenciar as partes e ajudam o usuário com o auto completar.

Além disso, esses editores avisam quando existe algum erro no código, permitem organizar melhor os arquivos do site e também facilitam com outras ferramentas no desenvolvimento. Por isso, os editores de código são mais eficientes e indicados para a criação de web, e principalmente para quem está começando.
